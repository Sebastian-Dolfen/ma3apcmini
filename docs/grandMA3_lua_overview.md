# grandMA3 Lua Overview (companion notes)

Light orientation notes for the grandMA3 Lua API surface, complementing `grandMA3_object_api.md` which carries the per-method Object-API detail.

Crawled from grandMA3 2.3 help. Sibling pages probed but **404** at the time of crawl: `lua_enums.html`, `lua_objecthandling.html`, `lua_genericfunctions.html`, `lua_keywords.html`. The help has reorganized those into the landing page and the two sub-APIs below.

## Lua Landing Page

Source: <https://help.malighting.com/grandMA3/2.3/HTML/lua.html>

Lua is a scripting language designed to support general procedural programming. It offers support for object-oriented programming, functional programming, and data-driven programming. Lua is implemented as a library, written in clean C (a common subset of ANSI C and C++).
For more information on scripting with Lua, see www.lua.org (https://www.lua.org/).
The grandMA3 software supports Lua version 5.4.6 and all the built-in standard libraries.
Use the Version keyword (keyword_version.html) in the command line to check which Lua version your device is currently running.
Besides the standard Lua libraries, there are some custom grandMA3 Functions.
A list of all the current grandMA3 functions can be created by using the HelpLua keyword (keyword_helplua.html). This keyword creates a file called "grandMA3_lua_functions.txt" in the gma3_library folder. Learn more about how to find this in the Folder Structure topic (fm_folder_structure.html).
This list separates the functions into two categories: Object-Free API (lua_objectfree.html) and Object API (lua_object.html). Individual functions are described in sub-topics in these two topics.
Lua code can be executed from plugins or directly in the command line using the Lua keyword (keyword_lua.html).
